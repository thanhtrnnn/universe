## Database Schema

### PostgreSQL (7 entities, TypeORM — auto-synced in dev)

| Entity | Table | Key Columns | Relations |
|--------|-------|-------------|-----------|
| User | `users` | id (UUID), email (unique), password (select:false), fullName, phone, studentId (unique), role (enum), isActive | → Enrollment[], Attendance[] |
| Course | `courses` | id (UUID), code (unique), name, credits, department, isActive | → Class[] |
| Class | `classes` | id (UUID), code, courseId FK, lecturerId FK, semester, room, maxStudents, **latitude**, **longitude**, isActive | → Course, User, Enrollment[], Schedule[] |
| Enrollment | `enrollments` | id (UUID), studentId FK, classId FK, status (enum), enrolledAt | UNIQUE(studentId, classId) → User, Class, Grade[] |
| Schedule | `schedules` | id (UUID), classId FK, dayOfWeek (0-6), startTime (time), endTime (time), room | → Class, Attendance[] |
| Attendance | `attendances` | id (UUID), scheduleId FK, studentId FK, sessionDate (date), status (enum), qrToken, gpsLatitude, gpsLongitude, **distance** (meters), scannedAt | → Schedule, User |
| Grade | `grades` | id (UUID), enrollmentId FK, type (enum), name, score, maxScore, weight (%) | → Enrollment |

### MongoDB (3 collections, Mongoose)

| Collection | Key Fields | Indexes |
|------------|-----------|---------|
| `messages` | senderId, receiverId, content, type (text\|image\|file), readAt | (senderId+receiverId+createdAt), senderId, receiverId |
| `notifications` | userId, title, body, type (attendance\|grade\|announcement\|system), data, isRead | (userId+isRead+createdAt) |
| `activity_logs` | userId, action, resource, resourceId, metadata, ipAddress, userAgent | (userId+createdAt), (resource+action) |

### Redis

| Key Pattern | Purpose | TTL |
|-------------|---------|-----|
| `refresh:<userId>` | JWT refresh token storage | 7 days |
| `qr:<signature>` | Dynamic attendance QR token | 5 seconds |
| `schedule:class:<classId>` | Cached class schedule | 1 hour |
| `schedule:student:<studentId>` | Cached student timetable | 1 hour |

## Running Locally (without Docker)

**Required services:**
1. PostgreSQL 15 on `localhost:5432` — `universe` database
2. MongoDB 7.0 on `localhost:27017` — `universe` database

**Optional services (degrade gracefully):**
3. Redis on `localhost:6379` — auth sessions + schedule cache

```bash
cd apps/server
cp .env.example .env   # edit credentials
npm install
npm run start:dev      # http://localhost:4000
```
